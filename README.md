# 1. 프로젝트 개요

## 프로젝트 소개
현대 기업의 성공적인 운영을 위해서는 통합된 정보 관리 시스템이 필수적입니다. GRID 프로젝트는 이러한 필요성을 인식하고, 기업이 휴가나 출퇴근 등의 인사 관리를 편리하게 수행하며, 동료 평가와 업적 평가 기능을 통해 업무 효율성을 향상시키는 것을 목표로 만들어졌습니다.
